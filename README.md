## Tips to write

### Write section Note
```
{% include code_note.html 
title='optional custom title'
content='text content note' 
%}
```
### Insert image
```
{% include code_image.html 
image='path/to/image.png'
title='title image'
target='_blank'
%}
```

### Link with target blank
sample: 
```
[text here](https://twitter.com/dagope){:target="_blank"}
```
